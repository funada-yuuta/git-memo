Gitメモ

git init

リポジトリの作成

git add

ステージング

git add .

リポジトリ内のすべてのファイルをステージング

git add ./sample.txt

sample.txtファイルのみステージング

git commit

コミット

git commit -m "コメント"

コメントの追加

git commit --amend -m "コメント"

直前のコミットのコメントの変更

git branch

ブランチの作成

git branch

ブランチ一覧と現在のブランチの表示

git branch develop

developブランチの作成

git branch -d feature

featureブランチを削除

git branch -m (master) main

ブランチ名の変更

git switch

作業中のブランチの切替

git switch develop

作業中のブランチをdevelopに切替

git switch -c feature

featureブランチを作成して切替

git reset

コミット・ステージングの取り消し

git reset

ステージング(git add)の取り消し

git reset --hard HEAD^

ひとつ前のコミットへ戻る(現在のステージング、変更、ひとつ前のコミットの内容の取り消し)

git reset HEAD^

ひとつ前のコミットへ戻る(変更点はそのまま)

git checkout

変更内容の取り消し

git merge

マージ

git merge develop

現在のブランチにdevelopをマージ

git diff

ステージングからの変更点を表示

git log

コミットのログを表示

git log --oneline

ログを1行で表示

git remote

リモートリポジトリの設定ん

git remote add origin git@github.com:funada-yuuta/git-memo.git

リモートリポジトリの追加

git remote -v

追加したリモートリポジトリの表示

git remote rm origin

リモートリポジトリの削除

git push

リモートリポジトリへプッシュ

git push origin main

mainブランチをリモートリポジトリへプッシュ
